# 心力自检表 · 后台配置说明

答完题后，数据会自动写入你的 **Google 表格**，手机、电脑都能打开表格查看「谁答了多少分」。

---

## 第一步：创建 Google 表格

1. 打开 [Google 表格](https://sheets.google.com)（需能访问 Google）
2. 新建空白表格，可命名为「心力自检答卷」
3. 菜单 **扩展程序 → Apps Script**
4. 删除编辑器里默认代码，把桌面上的 `xinli-zijian-backend.gs` **全部复制粘贴**进去
5. 点击 **保存**（项目名随意，如「心力后台」）

---

## 第二步：部署为 Web 接口

1. 在 Apps Script 右上角点击 **部署 → 新建部署**
2. 类型选 **网页应用**
3. 设置：
   - **执行身份**：我
   - **具有访问权限的用户**：**任何人**（必须，否则手机端无法提交）
4. 点击 **部署**，按提示授权
5. 复制生成的 **网页应用 URL**（形如 `https://script.google.com/macros/s/xxxxx/exec`）

---

## 第三步：填进答题页

1. 用 Cursor 打开 `xinli-zijian.html`
2. 找到顶部的 `CONFIG`，把 `SUBMIT_URL` 改成上一步复制的 URL：

```javascript
const CONFIG = {
  SUBMIT_URL: "https://script.google.com/macros/s/你的ID/exec",
};
```

3. 保存文件

---

## 第四步：发布公网链接（给别人手机答）

任选一种：

| 方式 | 操作 |
|------|------|
| **Netlify Drop** | [app.netlify.com/drop](https://app.netlify.com/drop) 拖入 `xinli-zijian.html` |
| **GitHub Pages** | 上传到仓库并开启 Pages |

把链接发到微信/群聊即可，**手机浏览器直接打开**。

---

## 第五步：查看后台答卷

打开你的 Google 表格，工作表 **「答卷记录」** 会自动出现，列包括：

| 列 | 含义 |
|----|------|
| 提交时间 | 答题时间 |
| 答卷编号 | 匿名编号（用于区分不同次提交，非姓名） |
| 总分 | 0–20 |
| 结果档位 | 心力较稳 / 略有透支 / 明显耗竭 |
| Q1–Q10 | 每题「是」或「否」 |

手机安装 **Google 表格 App** 可随时刷新查看新记录。

---

## 常见问题

**提交后表格没数据？**

- 检查 `SUBMIT_URL` 是否填对、是否以 `/exec` 结尾
- 部署时「具有访问权限的用户」必须是 **任何人**
- 修改脚本后需 **部署 → 管理部署 → 编辑 → 新版本 → 部署**

**国内访问 Google 困难？**

- 你可 VPN 查看表格；或改用 **腾讯文档智能表格 + 自动化 webhook**、**金数据**、**问卷星** 等国内表单（需另做一版对接）

**答卷是匿名的**

- 打开链接即可直接答题，无需填写姓名或手机
- 后台通过「答卷编号」区分每一次提交，无法自动对应到微信/小红书账号
